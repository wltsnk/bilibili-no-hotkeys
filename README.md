# Bilibili 键盘快捷键自定义

一个 Chrome/Edge 浏览器扩展，允许你在 B 站（bilibili.com）自由选择保留哪些键盘快捷键，其余全部禁用。

## 功能

- 默认保留空格（暂停/播放）、方向键（快进/快退/音量）、回车键
- 可在设置页面勾选需要保留的快捷键，分为六组：
  - 播放控制
  - 点赞投币弹幕
  - 全屏/退出
  - 页面/分P/滚动
  - 直播常用
  - 其他功能键
- 支持手动添加自定义按键
- 自动拦截非编辑区域的快捷键，不影响输入框和评论区打字
- Ctrl/Alt/Meta 组合键（复制粘贴等系统快捷键）不受影响
- 设置通过 Chrome 同步跨设备生效

## 安装

1. 下载或克隆本仓库
2. 打开浏览器扩展管理页面：`chrome://extensions/` 或 `edge://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹

## 使用

点击浏览器工具栏的扩展图标 → 选项，进入设置页面勾选/添加需要的按键。修改后刷新 B 站页面生效。
